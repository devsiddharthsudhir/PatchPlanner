# Demo Script (5–7 minutes)

1) Start with equal weights. Show baseline schedule.
2) Slide to **Risk-first**:
   - highlight KEV/high EPSS patches moving earlier
3) Slide to **Outage-first**:
   - show high-downtime/high change-risk patches getting deferred
4) Click a patch card:
   - read “why this patch” explanation
5) End with the point:
   - “Same backlog, different business reality => different optimal plan.”
